TemaTres-Vocabulary-Server by tematres

Web application for management formal representations of knowledge, like controlled vocabularies, taxonomies and thesauri

Key capabilities Scalable More than 1.000.000 terms in this server Service oriented architecture Flexible and powerful web services Compatible Export in any format: Skos-Core, Zthes, TopicMap, Dublin Core, MADS, BS8723-5, RSS, SiteMap, txt, SQL

Install

1. Unpack TemaTres in your web document root (i.e. /var/www/tematres/) 2. Configure MySQL database connection in /include/db.tematres.php 3. Open your browser in your tematres path installation and run Install script .
